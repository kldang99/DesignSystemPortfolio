Accessibility Considerations

Make sure to use an accessibility color checker tool to confirm that text has a contrast ratio of at least 4.5:1. The footer should work fully with a keyboard and use the footer tag for screen readers. The footer above uses dark text on a light background. If using a dark background, ensure the text is light enough to meet contrast requirements.
